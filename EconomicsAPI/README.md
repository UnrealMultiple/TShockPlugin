## 指令

| 语法                        |           权限           |   说明   |
| --------------------------- | :----------------------: | :------: |
| /bank add [玩家名称] [数量] |      economics.bank      | 添加货币 |
| /bank del [玩家名称] [数量] |      economics.bank      | 删除货币 |
| /bank pay [玩家名称] [数量] |    economics.bank.pay    | 转账货币 |
| /查询                       | economics.currency.query | 查询货币 |

# Back

- Authors: Megghy，肝帝熙恩
- 出处: TShock中文官方群聊
- It can return to the player’s last death location and allows for customizable cooldown time.

## Commands

| 语法    | Permission |            说明            |
| ----- | :--------: | :----------------------: |
| /back |    back    | Return to Death Location |

## Config

> Configuration file location：tshock/Back.en-US.json

```json5
{
  "CD": 20 //Cool Down Time
}
```

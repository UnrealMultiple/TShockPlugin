# BedSet 设置重生点

- Author: pang2333
- Use the recall item to set the respawn point, and teleport there upon logout or death

## Command

| 语法     |                   Permission                  |   说明  |
| ------ | :-------------------------------------------: | :---: |
| /shome | bed.spawn.set | 设置出生点 |

## Config

> Configuration file location：tshock/Bed.en-US.json

```json5
{
  "SpawnOption": {} //Spawn Point
}
```

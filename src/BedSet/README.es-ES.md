# BedSet 设置重生点

- 作者: pang2333
- 使用回城物品记录重生点，在退出及死亡后传送

## 指令

| 语法     |                       权限                      |   说明  |
| ------ | :-------------------------------------------: | :---: |
| /shome | bed.spawn.set | 设置出生点 |

## 配置

> 配置文件位置：tshock/Bed.zh-CN.json

```json5
{
  "重生点": {}
}
```

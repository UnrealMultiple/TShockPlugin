# Economics.Shop 商店插件

- 作者: 少司命
- 出处: 无
- 商店插件

> [!NOTE]\
> 需要安装前置插件：EconomicsAPI、Economics.RPG (本仓库)

## 指令

| 语法                                                                  |               权限               |   说明   |
| ------------------------------------------------------------------- | :----------------------------: | :----: |
| /shop buy [序号]  | economics.shop |  购买商品  |
| /shop list [页码] | economics.shop | 查看商品列表 |

## 配置

> 配置文件位置：tshock/Economics/Shop.json

```json5
{
  "最大显示页": 10,
  "商品列表": [
    {
      "商品名称": "test",
      "商品价格": 1,
      "等级限制": [],
      "进度限制": [],
      "执行命令": [],
      "商品内容": [
        {
          "物品ID": 1327,
          "数量": 0,
          "前缀": 0
        }
      ]
    }
  ]
}
```

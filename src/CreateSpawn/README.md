## 指令

| 语法      |    权限     |        说明        |
| --------- | :---------: | :----------------: |
| /cb set 1 | create.copy |   敲击建筑左上角   |
| /cb set 2 | create.copy |   敲击建筑右上角   |
| /cb save  | create.copy | 保存复制的建筑信息 |
| /create   | create.copy |      生成建筑      |

## 配置
> 配置文件位置：tshock/Create.json
```json
{
  "centreX": 145, //不要动
  "CountY": 178, //不要动
  "微调X": -21,
  "微调Y": 50
}
```

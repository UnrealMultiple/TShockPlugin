# ProgressBag 进度礼包

- 作者: 少司命
- 出处: 无
- 每当完成一个进度时可以领取一个礼包

## 指令

| 语法            |    权限     |    说明     |
|---------------|:---------:|:---------:|
| /bag r [礼包名称] |  bag.use  |   领取礼包    |
| /bag rall      |  bag.use  | 领取所有可领取礼包 |
| /bag list      |  bag.use  |  查看礼包列表   |
| /bag reset        | bag.admin |  重置领取礼包   |

## 具体进度请查看 ProgressType.cs 文件

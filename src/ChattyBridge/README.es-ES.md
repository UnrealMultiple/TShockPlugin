# ChattyBridge 聊天桥

- 作者: 少司命
- 出处: 无
- 让多个服务器之间可以互相聊天

## 指令

```
无
```

## 配置

> 配置文件位置：tshock/ChattyBridge.json

```json5
{
  "是否转发指令": false,
  "Rest地址": [
    "127.0.0.1:5000"
  ],
  "服务器名称": "玄荒",
  "验证令牌": "", //该令牌不是Rest令牌
  "消息设置": {
    "聊天格式": "[{0}]{1}: {2}",  // 0为服务器名 1为玩家名 2为消息
    "离开格式": "[{0}]{1}离开服务器",
    "加入格式": "[{0}]{1}加入服务器"
  }
}
```

# CriticalHit 打击文字提示

- 作者: White,Stone·Free，肝帝熙恩
- 出处: TShock官方群聊
- 在攻击NPC如果产生暴击，会弹出特殊的浮动文字（和伤害数字一样）
- 根据武器类型不同产生不同文字
- 可以设置即使不暴击也产生文字

## 指令

```
暂无
```

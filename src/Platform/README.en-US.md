# Platform 判断玩家设备

- 作者: Cai、cc04(维度)
- 出处: [github](https://github.com/ACaiCat/CaiPlugins)
- 判断玩家是用什么设备进来的（只能判断类型）

> [!NOTE]
> 此插件是用于作为前置插件供其他插件调用的(可用于聊天前缀、计分板判定等)\
> 目前支持区分的的PE、Stadia、XBOX、PSN、Editor、Switch、PC

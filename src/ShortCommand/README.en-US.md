# ShortCommand 简短指令

- 作者: GK、少司命、羽学
- 出处: QQ群（232109072）
- 这是一个Tshock服务器插件主要用于将插件或原生自带的Tshock命令映射成一个自己想要的命令
- 并可通过配置文件中的【阻止原始】彻底禁用一个命令，加了Reload重载配置文件方法
- 并配备了一个权限名【免检指令】使特定用户组不受【阻止原始】的影响

## 指令

| 语法 |  权限  |       说明      |
| -- | :--: | :-----------: |
| 无  | 免禁指令 | 插件不对其禁止使用原始命令 |

## 配置

> 配置文件位置：tshock/简短指令.json

```json5
{
  "命令表": [
    {
      "原始命令": "执行",
      "新的命令": "重置玩家数据",
      "余段补充": true,
      "阻止原始": true,
      "限制条件": 0,
      "冷却秒数": 0,
      "冷却共享": false
    },
    {
      "原始命令": "pskill list",
      "新的命令": "被动书店",
      "余段补充": true,
      "阻止原始": false,
      "限制条件": 0,
      "冷却秒数": 0,
      "冷却共享": false
    }
  ]
}
```

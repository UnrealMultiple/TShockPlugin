# ProgressRestrict 超进度检测

- 作者: 少司命 & 恋恋魔改三合一
- 出处: github
- 超进度检测
- 可对物品，弹幕，buff 进行超进度检测

> [!NOTE]\
> 需要安装前置插件：DataSync(本仓库)

## 指令

| 语法 |                             权限                             |        说明       |
| -- | :--------------------------------------------------------: | :-------------: |
| 空  |     progress.item.white    |   超进度物品检查白名权限   |
| 空  | progress.projecttile.white |   超进度弹幕检查白名权限   |
| 空  |     progress.buff.white    | 超进度 Buff 检查白名权限 |

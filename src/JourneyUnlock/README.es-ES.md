## 指令

| 语法                                |                    权限                   |    说明   |
| --------------------------------- | :-------------------------------------: | :-----: |
| /journeyunlock 或 /junlock 或 /i解锁" |   journeyunlock.unlock  | 给自己解锁物品 |
| /unlockfor 或 /unlockf 或 /g解锁玩家"   | journeyunlock.unlockfor | 给别人解锁物品 |

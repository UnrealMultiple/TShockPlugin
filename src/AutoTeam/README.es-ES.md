# AutoTeamPlus 更好的自动队伍

- 作者: 十七，肝帝熙恩
- 出处: 无
- 自动分配一个组的玩家到特定队伍

## 指令

| 语法              |                权限               |       说明      |
| --------------- | :-----------------------------: | :-----------: |
| /autoteam 或 /at | autoteam.toggle |    开关自动队伍功能   |
| 无               |            noautoteam           | 有该权限不会被自动分配队伍 |

## 配置

> 配置文件位置：tshock/AutoTeam.zh-CN.json

- 中文英文队伍名字参考如下：

| 中文  | English |
| --- | ------- |
| 无队伍 | none    |
| 红队  | red     |
| 绿队  | green   |
| 蓝队  | blue    |
| 黄队  | yellow  |
| 粉队  | pink    |

- 配置示例

```json5
{
  "开启插件": true, 
  "组对应的队伍": {
    "guest": "pink",
    "default": "蓝队",
    "owner": "红队",
    "admin": "green",
    "vip": "none"
  }
}
```

# CaiCustomEmojiCommand 自定义表情命令

- 作者: Cai
- 出处: 本仓库
- 懒得输一些长长的命令?试试自定义表情命令，将常用的命令转换为表情执行
- EmojiID可以在wiki(https://terraria.wiki.gg/zh/wiki/%E8%A1%A8%E6%83%85)上查询, 本插件不支持跳过权限检查, 命令需要标识符(/或者.)

## 指令

```
无
```

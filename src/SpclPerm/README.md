# SpclPerm 服主特权
- 作者: 羽学
- 出处: 无
- 根据名单进服或复活自动执行：无敌、BUFF、物品、命令、设置背包，
- 用户组名无视所有开关，开关只对特权名单有效，需关用户组权限直接删组名

## 指令

```
暂无
```

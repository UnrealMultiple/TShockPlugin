# ReFishTask

- Authors: onusai 羽学
- Source: [tshock-quest-fish-unlimited](https://github.com/onusai/tshock-quest-fish-unlimited)
- 当你完成渔夫任务时可无限接渔夫任务
- 配置项支持是否不更换任务鱼

## 指令

```
暂无
```

## Config

> Configuration file location：tshock/刷新渔夫任务.json

```json5
{
  "是否切换任务鱼（关掉就可以1种鱼刷1天）": true // Whether to change fisherman task 
}
```

# ReFishTask 刷新渔夫任务

- 作者: onusai 羽学
- 出处: [tshock-quest-fish-unlimited](https://github.com/onusai/tshock-quest-fish-unlimited)
- 当你完成渔夫任务时可无限接渔夫任务
- 配置项支持是否不更换任务鱼

## 指令

```
暂无
```

## 配置

> 配置文件位置：tshock/刷新渔夫任务.json

```json5
{
  "是否切换任务鱼（关掉就可以1种鱼刷1天）": true
}
```

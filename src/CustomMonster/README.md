# CustomMonster 自定义怪物血量

- 作者: GK阁下、羽学
- 出处: Q群232109072  [CustomMonster](https://github.com/1242509682/CustomMonster)
- 这是一个Tshock服务器插件主要用于：  
- 修改Tshock服务器的怪物， 
- 可自定义于怪物与BOSS的修改AI、修改血量、  
- 修改弹幕、修改掉落物、修改范围BUFF、  
- 修改随从怪物、拉取玩家、  
- 修改防御、修改无敌、修改免疫岩浆、  
- 修改免疫陷阱、控制指定怪物数量、NPC保护等等  

## 指令

| 语法      | 别名 |      权限       |     说明     |
|---------|:--:|:-------------:|:----------:|
| /ggw    | 无  | CustomMonster | 隐藏多余配置项并重载 |
| /smc    | 无  | CustomMonster |   召唤标识怪物   |
| /reload | 无  | CustomMonster |   重载配置文件   |

# EndureBoost 拥有指定数量物品给指定buff

- 作者: 肝帝熙恩
- 出处: 本仓库
- 当玩家背包药水或某些物品到达一定数量后，给与指定buff
- 进入服务器刷新，指令刷新，死亡刷新，回城刷新



## 指令

| 语法                                      |     权限      |      说明       |
|-----------------------------------------|:-----------:|:-------------:|
| /ebbuff 或 /ldbuff 或 /loadbuff 或 /刷新buff | EndureBoost | 立即刷新本插件buff状态 |

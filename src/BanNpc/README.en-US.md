# BanNpc

- Authors: GK
- Source: GK

## Commands

| 语法                                                               |         Permission         |             说明             |
| ---------------------------------------------------------------- | :------------------------: | :------------------------: |
| /bm add [ID] | bannpc.use |    Add NPC to block list   |
| /bm del [ID] | bannpc.use | Remove NPC from Block List |
| /bm list                                                         | bannpc.use |           查看禁止生成表          |

## Config

> Configuration file location：tshock/BanNpc.en-US.json

```json5
{
  "阻止怪物生成表": []
}
```

# Maptp 双击大地图传送

- 作者: Nova4334，肝帝熙恩
- 出处: 未知
- 有权限的玩家打开大地图后，双击去往所指定的位置
- 更近一步是否允许传送到实心方块


## 指令

| 指令 |      权限      |       说明        |
|----|:------------:|:---------------:|
| 无  | maptp.noclip | 允许双击传送(不包括实心方块) |
| 无  |    maptp     | 允许双击传送(包括实心方块)  |

暂无

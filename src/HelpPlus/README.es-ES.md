# HelpPlus 更好的帮助

- 作者: Cai
- 出处: 此仓库
- 修复/help指令无法使用的问题
- 为/help添加简短提示
- 为/help <命令> 可以看见更详细的命令帮助

## 指令

```
/help <页码> 查看命令列表
/help <命令> 查看命令的详细帮助
```

## 配置

> 配置文件位置：tshock/HelpPlus.json

{
  "简短提示开关": true,
  "简短提示对应": {

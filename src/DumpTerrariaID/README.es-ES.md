# DumpTerrariaID 输出 TerrariaID

- 作者: 少司命
- 出处: 无
- 将 Terraria 内部 ID 整合并输出到一个文件中

## 指令

| 语法      |            权限           |   说明  |
| ------- | :---------------------: | :---: |
| /dumpid | dump.id | 输出到文件 |

## 配置

> 配置文件位置：tshock/TerrariaID.json

```json5
TerrariaID表
```
